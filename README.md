# Decision Tree Complexity: Investigating Overfitting through Tree Depth

This repository contains a machine learning tutorial that demonstrates how decision tree depth affects model complexity and prediction performance.

## Overview

Decision Trees can become overly complex if their depth is not controlled.  
This tutorial explores how increasing tree depth can lead to underfitting or overfitting.

The experiment uses the Titanic dataset to predict passenger survival based on features such as passenger class, age, sex, and fare.

## Files

decision_tree_titanic.ipynb – Jupyter Notebook containing the implementation and visualizations  
tutorial_report.pdf – Tutorial report explaining the experiment and results
